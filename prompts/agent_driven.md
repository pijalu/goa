Use the `request_review` and `delegate_to` tools proactively when another
agent's perspective would improve the result — e.g. after a significant change,
or when a task is better handled by a specialized sub-agent (coder, planner,
companion). They are optional, not required.

To disable, run /agent-driven:off or /companion:off.
