Enter or exit plan mode. In plan mode, write and edit tools can only modify PLAN.md.

**Deprecation notice:** This legacy PLAN.md plan mode is superseded by the
`/plan` command which provides structured, event-sourced plan management
with built-in review and sequential execution. The legacy tool remains
functional but new workflows should use `/plan` instead.
