## Swarm Mode Ended

Swarm Mode has ended. You are no longer required to follow the Swarm Mode workflow.

The user's next request is likely to be a regular request that does not need the `agent_swarm` tool. If the request still benefits from parallel subagents, you may call `agent_swarm`, but decide from the new request itself rather than the ended Swarm Mode workflow.
