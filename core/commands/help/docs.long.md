Usage: /docs:topic

List all available documentation, or show a specific document.

Embedded documentation files cover every aspect of Goa:
  AGENTIC-SDK      — How Goa wraps the agentic SDK
  ARCHITECTURE     — Full system architecture
  COMMANDS         — Complete command system reference
  CONFIGURATION    — Configuration cascade and settings
  DEVELOPMENT      — Development guide
  IMPLEMENTATION-ROADMAP — M10–M30 roadmap
  PROFILES         — Agent profiles
  SETUP            — Installation and setup
  SKILLS           — Skills system
  TOOLS            — Tool system reference
  TUI              — TUI layout and usage

Examples:
  /docs              List all available documents
  /docs:ARCHITECTURE Show the architecture documentation
  /docs:TOOLS        Show the tools reference
  /docs?             Short help (this message)
  /docs??            Long help with details
