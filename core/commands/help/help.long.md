Usage: /help:command

Show a list of all available commands, grouped by category.
If a command name is given, show detailed help for that command.

Examples:
  /help          List all commands
  /help:mode     Show help for the /mode:command
  /help?         Short help (alias)
  /help??        Long help with examples
