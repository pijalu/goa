Usage: /profile[:coder|:planner|:reviewer|:<custom>]

/profile is an alias for /mode. It exists for backward compatibility.
Use /mode for all mode operations.

Without arguments, opens an interactive mode picker. With a mode name,
switches directly to that mode.

Examples:
  /profile          Show mode picker
  /profile:coder    Switch to coder mode
