Usage: /hotkeys

Display every keyboard shortcut grouped by category (navigation, editing,
application actions). Shortcuts come from the default keybinding set; slash
commands accept completion candidates with Tab.

Examples:
  /hotkeys     Show the shortcut table
  /hotkeys?    Short help (this message)
