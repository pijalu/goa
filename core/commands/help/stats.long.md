Usage: /stats[:turn-number]

Show detailed token usage breakdown per turn.
With a turn number, show a detailed tree for that turn.

Examples:
  /stats        Show per-turn overview
  /stats:3      Show detailed breakdown for turn 3

Aliases: /tokens, /tok
